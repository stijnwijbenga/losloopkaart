# Losloopkaart

Een persoonlijke kaart-app met alle hondenlosloopgebieden in Nederland, inclusief filters op zwemwater, provincie, type en meer. Gebieden met een seizoensregel worden apart gemarkeerd zodat je weet welke je even moet nakijken.

De app werkt als website en is op je iPhone toe te voegen als eigen app-icoon — zonder appstore, gratis te hosten.

Data afkomstig van [doggydating.com](https://www.doggydating.com).
